Хоррор-Адвенчур Blindture

Проект созданный за 3 дня во время участия в Gamejam. Хоррор приключение в пещере за человека, ориентирующегося только на звуковые волны, как летучая мышь. Цель - искать себе путь и выбраться невредимым.

Как геймдизайнер придумал лор игры, основные механики. Вовремя этого искал референсы, когда привлекла модель пещер из Penumbra. Упор в игре делался на звуки, из объемность и правильную передачу чувств.

Моя роль: Геймдизайнер, разработчик

Аркадная гонка Nightdrive

Nightdrive - гоночная аркада для пк, которая разработанна на Unreal Engine 5.5. Целью было исследование жанра гонок и реализации хорошей физики автомобиля.

Разработка проекта началась с составления дизайн-документа, в котором были определены стиль аркадной гонки, ключевые игровые режимы и анализ конкурентов. Это позволило команде сформировать единое видение проекта.

Физика была реализована с применением физических формул на основе блюпринтов. Была использована модульная система для отдельной реализации коробки передач, а также подход Data-Driven Design для создания таблицы данных автомобиля, основанной на структурах данных для каждого из его состояний(мотор, коробка, подвеска и т.д.)

Также для передачи чувства скорости были реализованы методы изменений камеры в зависимости от скорости. И специально добавлены аркадные механики для оптимизации управления через клавиатуры, что необходимо для аркадных гонок на пк.

В качестве дизайнера интерфейсов был сверстан адаптивный HUD, спроектированный в Figma. Создан интерфейс тюнинга с специальными подсказками, вводящими игроков в суть изменяемых параметров.

Моя роль: Геймдизайнер, разработчик физики, UI/UX

p.wheel — симуляция физики автомобиля

Проект симулятора автомобиля на основе физики Nightdrive. Возможность настройки всех параметров физического состояния автомобиля, управления деталями и настройки чип тюнинга. Проект реализуется на Unreal Engine 5.5. Все результаты по проекту будут выкладываться на YouTube канал.
