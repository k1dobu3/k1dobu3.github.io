О чём наш проект?

Бот предоставляет возможность получать новости сайта Habr/develop в телеграмме. Работа с сервисом представляет из себя следующие шаги:
1.Регистрация в боте с указанием некоторых ваших данных для некоторых возможностей в будущем
2.Выбор представленных категорий новостей
3.Автоматическая получение статей по выбранным темам, а также выдача последних новостей сайта по запросе пользователя

На данном этапе в боте предоставлены три новостных группы:
1. Frontend - все статьи по тегам фронтенда
2. Django - все стать по тегам django
3. Develop - все новости раздела разработки

Как использовать бота?

Пользоваться нашим "инструментом" легко. Для этого надо выполнить немного действий:
1. Перейти по ссылке в нашего бота и выполнить команду /start
2. Пройти небольшую регистрацию
3. Выбрать категории новостей, которые вы хотели бы получать
4. Получать новостную рассылку, либо подборку последних новостей по вашему запросу
